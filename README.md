*NOMBRE DEL PROYECTO*: Proyecto_Turismo_Llanquihue

*DESCRIPCION DEL PROYECTO*

Este sistema busca representar y organizar la información de la empresa Turismo Llanquihue, 
permitiendo modelar de forma estructurada los datos de las personas vinculadas a la organización. 
Para ello, se utilizan clases orientadas a objetos que almacenan información como nombre, RUT y dirección, 
aplicando principios fundamentales de la Programación Orientada a Objetos. tales como 
estructura modular basada en clases, permitiendo representar Personas y guías turísticos 
mediante relaciones de composición. Además, incorpora validación de datos mediante excepciones personalizadas
para garantizar la integridad de la información ingresada.
